# BEAT 2.0.0a2 — frozen astronomer-pilot build

This prerelease incorporates the completed powered MUSE and NIRSpec gates,
bounded MIRI gate, NGC 1365 broad-H-alpha recommendation, and synthetic 1D
survey operational regression.

Download `BEAT-2.0.0a2-astronomer-pilot-20260720.zip`. The archive contains an
installable wheel, matching source, all 66 passing tests, portable examples,
instrument documentation, validation summaries, bounded pilot instructions,
and a structured feedback form.

Scientific use remains limited to the supported domains stated in the bundled
validation reports. Important limits include NIRSpec G235H effective component
S/N >= 10, conditional MIRI Gaussian-LSF support, representative rather than
dataset-matched MUSE LSF calibration, and a one-broad-component default for
routine broad-H-alpha fitting.

This is alpha software and should be marked as a GitHub prerelease.
